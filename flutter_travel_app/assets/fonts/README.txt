Place the following font files here (downloaded from Google Fonts):
- NotoSans-Regular.ttf
- NotoSans-Medium.ttf
- NotoSans-SemiBold.ttf
- NotoSans-Bold.ttf
- NotoSansMono-Regular.ttf
- NotoSansMono-Bold.ttf
- Roboto-Regular.ttf
- Roboto-Bold.ttf

You can download them manually from:
https://fonts.google.com/specimen/Noto+Sans
https://fonts.google.com/specimen/Noto+Sans+Mono
https://fonts.google.com/specimen/Roboto

Or run the PowerShell script `fetch_fonts.ps1` that will be added to automate this.
